# Тема работы: Приложение для отслеживания привычек

Основные функции:

1\. Приложение должно работать с отслеживанием привычек. Включая отказ от вредных и прививание новых. 

2\. Эскизы прикреплены в архиве

3\. Сценарии использования:

а)Добавление привычки в свою жизнь (ставим цель и ежедневно отмечаем выполнение)

б)Датируемые заметки по привычкам

в)Отказ от привычки (таймер увеличивается до прерывания)

г)Мотивационные фразы на главном экране

д)Возможность делиться своими достижениями
